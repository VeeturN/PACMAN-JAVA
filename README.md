# Pacman Java Edition 🕹️

A professional, classic implementation of the legendary **Pacman** arcade game, built from scratch using Java. This project recreates the authentic retro experience with smooth movement, ghost AI, and a classic scoring system.

---

## 🚀 Features
* **Classic Mechanics**: Precise movement, pellet consumption, and power-up logic.
* **Ghost Intelligence**: Implementation of unique movement patterns for different ghosts.
* **Level Progression**: Increasing difficulty as you clear the board.
* **Scoring System**: Points for pellets, fruits, and hunting ghosts under power-up effects.
* **Retro Graphics**: Clean UI built with Java Swing/AWT.
* **Lives & Game Over**: Traditional arcade life system.

---

## 🛠️ Requirements
To run this project, you need:
* **Java Development Kit (JDK)**: Version 11 or higher.
* Any Java-supported IDE (IntelliJ IDEA, Eclipse, VS Code) or a Terminal.

---

## 🏗️ Installation & Execution

1. **Clone the repository**:
   ```bash
   git clone [https://github.com/VeeturN/PACMAN-JAVA.git](https://github.com/VeeturN/PACMAN-JAVA.git)
   cd PACMAN-JAVA
   javac src/*.java -d bin
   java -cp bin Main

---

## 🎮 Controls
Use your keyboard to navigate Pacman through the maze:
* Arrow Keys / WASD: Move Up, Down, Left, Right
* ESC: Exit to Menu / Quit
